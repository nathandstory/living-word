# Changelog

## 0.1.0 - 2026-05-25

Initial public release.

### Added

- Craftable Bible item.
- Scripture reading GUI with search, highlights, and translation support.
- Bundled BSB, KJV, and WEBP Bible text data.
- On-demand Bible audio streaming and local caching.
- KJV AudioTreasure provider manifest.
- BSB HelloAO provider manifests for David, Souer, and Hays narrators.
- WEBP David Williams public-domain audio provider manifest.
- Generated timing sidecars for supported KJV and BSB narrators.
- Scripture Disc selection and jukebox playback.
- Lectern listening station behavior with floating verse display.
- Session synchronization infrastructure for multiplayer listening.
- Shofar item, recipe, sound, and advancement.
- Basic advancement set for Living Word progression.

### Notes

- Chapter audio is streamed and cached by clients; large chapter audio files are not bundled in the mod jar.
- This is an early public release. Test on a backup world before using it on a public server.
